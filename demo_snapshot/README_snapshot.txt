High School English Writing Grading Assistant（高中英语作文批改助手） static demo snapshot
Version: 0625b

Usage: open index.html in a browser after cloning or downloading this repository.

The static HTML snapshot supports four-page navigation and keeps CSV / Excel download links on the final page. It is intended for visual demonstration only. The live Streamlit application remains the source of truth for the actual front-end experience.

The sample answer sheets included in this snapshot are anonymized or self-created demonstration samples. They do not contain student names, school names, class information, contact details, examination IDs, or other personally identifiable information.
