# Sample Data

This folder contains anonymized demonstration data.

- `answer_sheets/`: 12 anonymized or self-created demonstration answer-sheet images.
- `sample_prompt.txt`: sample writing prompt.
- `scoring_rubric.txt`: default scoring rubric used in the demo.
- `sample_output.csv` / `sample_output.xlsx`: exported sample results.
- `sample_student_*`: example OCR text and model feedback.

The sample answer sheets included in this repository are anonymized or self-created demonstration samples. They do not contain student names, school names, class information, contact details, examination IDs, or other personally identifiable information. They are provided only to demonstrate the application workflow.
